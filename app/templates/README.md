# 使用说明

## 目录说明


```
|- dist                             #最终压缩的目录
  |- widget.zip                     #组件模板压缩包
|- src
  |- widget.css                     #组件样式
  |- widget.handlebars              #组件模板
  |- widget.json                    #组件数据

```


## 运行说明

1. 运行 grunt 或 grunt server 启动服务

2. 运行 grunt build 打包 src 下的文件到 dist 目录下
